# InsightSphere-Data-Analytics-Dashboard
🚀 InsightSphere: Data Analytics Dashboard  InsightSphere is an interactive data analytics web application built using Streamlit that enables users to upload datasets and instantly generate insights, visualizations, and a complete analytical dashboard.<br>
It simplifies data exploration by providing automated analysis, customizable charts, downloadable reports— all in a user-friendly interface. 
## ✨ Features
📂 Upload CSV datasets<br>
🔍 Data preview and statistical summary<br>
💡 Automated insights and recommendations<br>
📊 Interactive and customizable charts:<br>
Bar Chart<br>
Line Chart<br>
Area Chart<br>
Histogram<br>
Pie Chart<br>
Column Chart<br>
🔥 Correlation Heatmap<br>
📈 Dashboard with key metrics and multiple visualizations<br>
⬇ Download:<br>
Processed data (CSV)<br>
Full dashboard (PNG/HTML)<br>
# 🛠️ Tech Stack
Frontend & App Framework: Streamlit<br>
Data Processing: Pandas<br>
Visualization: Plotly<br>
