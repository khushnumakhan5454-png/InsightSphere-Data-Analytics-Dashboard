# InsightSphere-Data-Analytics-Dashboard
🚀 InsightSphere: Data Analytics Dashboard  InsightSphere is an interactive data analytics web application built using Streamlit that enables users to upload datasets and instantly generate insights, visualizations, and a complete analytical dashboard. It simplifies data exploration by providing automated analysis, customizable charts, downloadable reports— all in a user-friendly interface. 
## ✨ Features
📂 Upload CSV datasets
🔍 Data preview and statistical summary
💡 Automated insights and recommendations
📊 Interactive and customizable charts:
Bar Chart
Line Chart
Area Chart
Histogram
Pie Chart
Column Chart
🔥 Correlation Heatmap
📈 Dashboard with key metrics and multiple visualizations
⬇ Download:
Processed data (CSV)
Full dashboard (PNG/HTML)
# 🛠️ Tech Stack
Frontend & App Framework: Streamlit
Data Processing: Pandas
Visualization: Plotly
## 🚀 How to Run Locally
# 1. Clone the repository
 git clone https://github.com/khushnumakhan5454-png/InsightSphere-Data-Analytics-Dashboard/new/main?readme=1
 cd InsightSphere: Data analytics Dashboard
# 2. Install dependencies
 pip install -r requirements.txt
# 3. Run the application
 streamlit run lab.py
